Instagram Gallery is a Wordpress plugin designed to provide a beautiful and highly customized Instagram Gallery with amazing and innovative effects.

== Description ==

You can display the information from your account. Image and video Gallery has CSS3 Hover Animation & Display with Lightbox. This plugin is an advanced plugin with a list of option for adding and editing Instagram. It is fully responsive. The Product Allow you to add unlimited shortcode to you post/page. You can add multiple instagram from multiple accounts.


Using **[IGF id=123]** shortcode, you can publish gallery in any Page or Post in your WordPress sites.

== Installation ==

1. Upload the entire **instagram-gallery** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Create new gallery, Use **[IGF id=123]** shortcode to publish your gallery into any page or post.
